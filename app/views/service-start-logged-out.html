About the identity management (IDM) service

This service will help our users to create a single set of sign-in credentials that they can use to access one or more Defra services. This will save users time and effort.

We will store important information about the user (and their business, if relevant) in the Dynamics database so that they do not have to re-enter it each time they want to use a new online service.

It will also allow them to give other people (for example colleagues or agents they work with) access to one or more of the online services they use.

The registration user journey flow:

  1. Starts on a specific service page (built and owned by the relevant service).
  2. Takes the user into the IDM service to select an account type.
  3. Asks the user to enter their existing Government Gateway account details, or leads them to the new SCP service to create a new Government Gateway ID.
  4. Brings the user back to the IDM service to provide details about themself and, if relevant, their business.
  5. Sends the user back into their chosen service to complete the required registration process.
